Lenny: The Library-in-a-Box

Lenny is an plug-and-play, open-source, Library-in-a-Box that empowers libraries to preserve, own, and lend digital books on their own terms.

Get Started

Quick Installation

Get Lenny running in minutes with our simple one-line installation command

Terminal Command
curl -fsSL https://raw.githubusercontent.com/ArchiveLabs/lenny/refs/heads/main/install.sh | sudo sh

Downloads Dependencies

Automatically installs Docker and all required components

Quick Setup

Configures Lenny with sensible defaults for immediate use

Production Ready

System Requirements

Requires a Linux system with curl and sudo access. Compatible with Ubuntu, Debian, CentOS, and most modern distributions.

Why Lenny?

Free, open source, built by nonprofits for the public good.

Lenny is an open prototype, incubated by Archive Labs and the team behind Open Library at the Internet Archive. It's designed as a public good: a free digital circulation system for libraries that want to preserve, lend, and serve their communities on their own terms.
